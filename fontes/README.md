# Guia de Estudos Salesforce com NotebookLM

## 📌 Contexto

Este projeto foi desenvolvido como parte de um desafio prático da DIO (Digital Innovation One), com o objetivo de utilizar o Google NotebookLM como ferramenta de aprendizagem ativa.

O tema escolhido foi **Salesforce**, plataforma líder mundial em CRM (Customer Relationship Management), amplamente utilizada por empresas para gerenciar clientes, processos comerciais e relacionamento com o mercado.

---

## 🎯 Objetivos

Este caderno temático possui os seguintes objetivos:

- Consolidar conhecimentos sobre Salesforce;
- Organizar conteúdos estudados em um único local;
- Utilizar Inteligência Artificial para auxiliar nos estudos;
- Documentar o processo de aprendizagem;
- Criar um material de consulta para futuras revisões.

---
